This folder contains screenshots demonstrating the deployment of a static website using Amazon S3 and Amazon CloudFront.

## Project Summary

This project demonstrates:

* Amazon S3 Static Website Hosting
* Amazon CloudFront CDN Configuration
* Origin Access Control (OAC)
* S3 Bucket Policy Management
* Secure Content Delivery
* HTTPS-enabled Website Hosting

The website is hosted on Amazon S3 and securely delivered through Amazon CloudFront for better performance, scalability, and security.
